# コンプライアンス・表記ガイド

## 必須表記

1. **非公式である旨** — エポスおよび各カード会社の公式サイトではないこと
2. **試算の免責** — 参考値であり保証しないこと
3. **#PR** — アフィリエイトリンク設置時は申込導線近くに明示
4. **最終確認日** — 各カード JSON の `last_updated` と公式 URL を併記

## 表現上の注意

- 「最安」「絶対お得」などの断定表現は避ける
- 「条件を満たす場合」「試算上」などの限定表現を使う
- 還元率は「最大◯%」ではなくユーザー条件に基づく試算値を表示

## ステマ規制（2023年10月〜）

景品表示法のステマ規制に基づき、広告・PR に該当するリンクには `#PR` またはそれに準ずる表記を行う。
